About Me

A hybrid of engineer, builder, designer, and storyteller based in Zulte, Belgium.

Professional Journey

Hi, I'm Jan Blomme, a hybrid of engineer, builder, designer, and storyteller based in Zulte, Belgium. I'm currently studying Multimedia Creative Technologies at HOWEST, with a strong specialization in AI engineering and XR development, and I've completed an exchange at Portsmouth University in the Faculty of Creative and Cultural Industries.

But degrees only tell half the story. What really defines me is how I build things that actually work. I don't just write code. I design systems, build devices, tune interfaces, and connect every layer from microcontroller firmware to intuitive user experience.

Whether it's a tactile XR controller made from wood and ball bearings, a local-first study suite with full offline support, or a fully integrated IoT cocktail machine, I'm constantly fusing creativity and engineering into real, usable tools.

Key Projects

Bridging the digital and physical worlds through innovative hardware and software solutions.

OrbitalPlay Controller (2025)

An open-source XR hardware controller using omnidirectional ball-on-bearing tracking, powered by a Seeed XIAO RP2040. Features C++ firmware, paddle input + game logic in JS + Canvas. Fully documented with CAD files, firmware, BOM on GitHub. Designed for tactile realism and immersive demos.

C++ JavaScript Canvas Hardware XR

Smart Cocktail Shaker (2024)

An IoT drink mixer with a custom Raspberry Pi touchscreen UI. Features motor-controlled liquid dispensing and Python GPIO logic. Designed for demo usability, with physical buttons, screen UI, and automation. Combines hardware engineering with intuitive software design.

Python Raspberry Pi GPIO IoT Hardware

NMBS PowerApp (2023)

Hackathon runner-up: onboarding app for the Belgian rail company. Built in Microsoft PowerApps with mobile-first, accessibility-first approach. Optimized for internal enterprise use with focus on user experience and efficient workflow integration.

PowerApps Mobile-First Accessibility Enterprise UX

Student Tools Suite (Ongoing)

A fully local, offline-first productivity toolkit for students. Built in HTML/CSS/JS with Tailwind and IndexedDB. Includes Flashcard app, Infinite Whiteboard, Pomodoro Timer, Note-taking with Markdown, Quiz Generator (CSV input), and more. Unified design system and storage logic. No accounts, no cloud dependencies, just performance and privacy.

HTML/CSS/JS Tailwind IndexedDB Offline-First Privacy

Relivem App (In Progress)

A nostalgic daily photo archive app that turns memories into timelines. Features Duolingo-style vertical scroll layout, SQLite local storage with export metadata (title, location, people, date, logo), and Firebase for login and syncing. Built in Dart + Flutter with tagging, exporting, and smart search. Designed for private archiving, not social media.

Dart Flutter SQLite Firebase Mobile
